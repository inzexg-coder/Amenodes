# Amenodes Roadmap

Единый список из 143 пунктов: ноды, дизайн, функции. От простого к сложному.

---

**1.** Более плавный `transition: all 0.3s ease` вместо жёсткого `0.1s` на все ноды

**2.** При ховере на нод — лёгкое увеличение `scale(1.02)` + glow

**3.** При ховере на ребро — подсветка ярким цветом + утолщение

**4.** При ховере на порт — `scale(1.5)` + яркая вспышка

**5.** При drag нода — увеличенная тень и `scale(1.03)`

**6.** Курсор при перетаскивании связи — crosshair

**7.** `Ctrl+C` / `Ctrl+V` — копировать/вставить выделенный нод

**8.** `Ctrl+D` — дублировать нод

**9.** `Delete` / `Backspace` — удалить выделенный нод (глобально, не только через меню)

**10.** `Ctrl+Z` / `Ctrl+Y` — undo/redo (сейчас только кнопками)

**11.** `+` / `-` — приблизить/отдалить

**12.** Стрелки — перемещать выделенный нод на 1px, Shift+стрелка на 10px

**13.** `Home` / `F` — zoom to fit весь граф

**14.** `F2` — переименовать выделенный нод

**15.** `Tab` — открыть меню добавления нода в позиции курсора

**16.** **CalcNode → режим `instrument_C2`** — Δ = C/2

**17.** **CalcNode → режим `instrument_C_sqrt12`** — Δ = C/√12

**18.** **CalcNode → режим `instrument_class`** — Δ = (k·X_lim)/100

**19.** **CalcNode → режим `instrument_digit`** — Δ = 1 разряд

**20.** **CalcNode → режим `relative_error`** — ε = σ/x

**21.** Анимация появления нода: `scale(0.8) → scale(1)` + fade

**22.** Анимация удаления нода: `scale(1) → scale(0.5)` + fade

**23.** Splash-экран: логотип вращается + частицы на фоне

**24.** Canvas открывается: splash разлетается, canvas выезжает снизу

**25.** Toolbar при загрузке — элементы появляются последовательно (stagger)

**26.** Status bar — slide up при появлении

**27.** Sidebar открывается/закрывается — smooth slide + иконка вращается

**28.** Node menu выезжает из кнопки «+» с scale + backdrop blur

**29.** Dropdown в CalcNode — fade + slide при открытии

**30.** При создании связи — линия «рисуется» (stroke-dashoffset)

**31.** При удалении связи — линия исчезает с fade

**32.** Кнопки в тулбаре — glow + scale + ripple при наведении

**33.** При mark IMPORTANT — волна синего свечения

**34.** При изменении значения — жёлтая вспышка на 300ms

**35.** Output-таблица — строки появляются с задержкой

**36.** Undo/Redo — кнопки «отскакивают» при нажатии

**37.** Multi-select: `Shift+Click` на нескольких нодах

**38.** Rectangle select — drag по пустому месту выделяет рамкой

**39.** **CalcNode → режим `student_t`** — t(n, α)

**40.** **CalcNode → режим `total_error`** — σ = t·√(σ²_rand + σ²_inst)

**41.** Ноды при загрузке сохранённого графа — stagger появление

**42.** Snap to grid — пружинка при прилипании (overshoot + settle)
