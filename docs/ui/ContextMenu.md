# `ContextMenu`

## ОПИСАНИЕ

`ContextMenu` – класс, реализующий контекстное меню для узлов графа. Меню вызывается **правым кликом** на узле.

**Ключевые особенности:**
- Отметка узлов как «важных» (визуальное выделение)
- Снятие отметки важности
- Поддержка динамического обновления переводов

## ЗАВИСИМОСТИ

```javascript
import { i18n, t } from '../i18n/LanguageManager.js';
```

| Импорт | Назначение |
|--------|------------|
| `i18n`, `t` | Система интернационализации для локализации текстов меню. |

# КЛАСС CONTEXTMENU

## Конструктор

```javascript
constructor(graph: Graph, renderer: DomRenderer, history: History, viewport: Viewport)
```

Создаёт экземпляр контекстного меню, привязанный к конкретным компонентам приложения.

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `graph` | `Graph` | Экземпляр графа. |
| `renderer` | `DomRenderer` | Экземпляр рендерера. |
| `history` | `History` | Экземпляр системы истории (undo/redo). |
| `viewport` | `Viewport` | Экземпляр области просмотра. |

**Инициализируемые свойства:**

| Свойство | Тип | Начальное значение | Описание |
|----------|-----|--------------------|----------|
| `this.graph` | `Graph` | параметр | Ссылка на граф. |
| `this.renderer` | `DomRenderer` | параметр | Ссылка на рендерер. |
| `this.history` | `History` | параметр | Ссылка на систему истории. |
| `this.viewport` | `Viewport` | параметр | Ссылка на область просмотра. |
| `this.currentMenu` | `HTMLElement \| null` | `null` | Текущее отображаемое DOM-элемент меню. |
| `this.currentSourceId` | `number \| null` | `null` | ID узла, для которого открыто меню. |

- Подписывается на изменения языка через `i18n.subscribe()`. При смене языка текущее открытое меню автоматически перестраивается с новыми переводами.

# Публичные методы

## show(x, y, sourceId)

```javascript
show(x: number, y: number, sourceId: number): void
```

Отображает контекстное меню в указанных экранных координатах для заданного узла.

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `x` | `number` | Горизонтальная координата (в пикселях, относительно окна браузера) для левого верхнего угла меню. |
| `y` | `number` | Вертикальная координата (в пикселях, относительно окна браузера) для левого верхнего угла меню. |
| `sourceId` | `number` | Идентификатор узла, для которого вызывается меню. |

**Структура меню:**

1. Пункт **"Mark IMPORTANT node"** – выделить узел как важный
2. Пункт **"Unmark IMPORTANT node"** – снять выделение важности

**Пример:**

```javascript
const menu = new ContextMenu(graph, renderer, history, viewport);
menu.show(200, 300, 5); // открыть меню для узла с ID=5 в позиции (200, 300)
```

## toggleImportant(node, important)

```javascript
toggleImportant(node: Node, important: boolean): void
```

Устанавливает или снимает флаг важности узла.

**Параметры:**

| Имя | Тип | Описание |
|-----|-----|----------|
| `node` | `Node` | Целевой узел. |
| `important` | `boolean` | `true` для отметки как важного, `false` для снятия отметки. |

**Действия:**
- Изменяет свойство `node.important`.
- Вызывает `renderer.updateNodeClass(node)` для обновления CSS-класса.
- Вызывает `renderer.render()` для перерисовки.
- Вызывает `history.save()` для сохранения состояния.

## close()

```javascript
close(): void
```

Закрывает текущее открытое контекстное меню.

**Действия:**
- Удаляет DOM-элемент меню из документа.
- Устанавливает `this.currentMenu = null` и `this.currentSourceId = null`.

# Приватные методы

## addMenuItem(menu, text, onClick)

Добавляет стандартный пункт меню.

**Параметры:**
- `menu` – DOM-элемент, в который добавляется пункт.
- `text` – текст пункта меню.
- `onClick` – функция, вызываемая при выборе пункта (после чего меню автоматически закрывается).

# ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Базовое использование

```javascript
import { ContextMenu } from './ui/ContextMenu.js';

const contextMenu = new ContextMenu(graph, renderer, history, viewport);
contextMenu.show(500, 400, 3);
```

### Интеграция с рендерером

```javascript
// В DomRenderer.js
showMenu(x, y, sourceId) {
  if (!this.contextMenu) {
    this.contextMenu = new ContextMenu(this.graph, this, this.history, this.viewport);
  }
  this.contextMenu.show(x, y, sourceId);
}
```

### Закрытие меню извне

```javascript
window.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') {
    contextMenu.close();
  }
});
```

## ЗАМЕЧАНИЯ

- **Локализация:** Меню автоматически перестраивается при смене языка.
- **Позиционирование:** Меню позиционируется абсолютно относительно окна браузера (использует `position: fixed` в CSS).
- **Закрытие:** После выбора любого пункта меню закрывается автоматически.
- **Навигация:** Класс не управляет фокусом клавиатуры – навигация по меню возможна только мышью.
- **Создание узлов:** Для добавления новых узлов используйте кнопку **"+" (Add node)** на верхней панели инструментов. Контекстное меню предназначено только для управления важностью существующих узлов.
